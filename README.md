### 🛠️ Prova Técnica: Lógica de Agente Financeiro
Abaixo, demonstro um exemplo de como a IA auxilia na estruturação de lógica para o meu projeto **CAIXA Família**:

```javascript
// Estrutura de cálculo processada via Prompt Engineering
const calcularSaldo = (receitas, despesas) => {
  const total = receitas.reduce((a, b) => a + b) - despesas.reduce((a, b) => a + b);
  return `Saldo processado pelo Segundo Cérebro Tech: R$ ${total}`;
};

console.log(calcularSaldo([5000, 1200], [1500, 800]));
